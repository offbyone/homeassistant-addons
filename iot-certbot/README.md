# Home Assistant Add-on: IoT Thing + Certbot

_Register your Home Assistant as an IoT Thing and then use certbot + route53 to get certificates._

This is based on ["Using AWS IoT to provision an IAM role for home lab devices"](https://offby1.website/posts/automating-letsencrypt-route53-using-aws-iot.html)
